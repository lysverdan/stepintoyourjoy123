## ğŸ’³ Payment Integration

### Ko-fi Setup
1. Sign up at https://ko-fi.com
2. Upload your digital products
3. Get your Ko-fi link (e.g., `https://ko-fi.com/yourname`)
4. Replace `KOFI-LINK-HERE` in your HTML files with your actual link

### PayPal Setup
1. Create PayPal Business account (or use personal)
2. Get your PayPal.me link OR create payment buttons at https://www.paypal.com/buttons
3. Replace `PAYPAL-LINK-HERE` in your HTML files with your actual link

### Example Payment Buttons (for product pages):

## ğŸ¨ Customization

### Colors
Edit the CSS variables in `styles.css`:
```css
:root {
    --primary: #d4a373;        /* Warm gold - change this */
    --primary-dark: #b8885a;   
    --secondary: #8b7a6a;      
    /* etc. */
}
```

### Fonts
Current fonts are:
- **Display/Headers:** Cormorant Garamond (elegant serif)
- **Body:** Lato (clean, readable)

To change, edit the Google Fonts link in `index.html` and update the CSS variables.

### Adding Your Photos
1. Create an `images` folder in your repository
2. Upload your photos
3. Add them to HTML like this:
```html
<img src="images/your-photo.jpg" alt="Description">
```
